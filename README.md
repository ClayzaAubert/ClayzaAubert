## `> whoami`
<img align="right" src="https://avatars.githubusercontent.com/u/92805294?v=4" width="310" /><br/>

I'm just a **wanderer** — no fancy title, no strict path.

I float between stacks and build whatever sparks curiosity.

Sometimes it's a clean REST API.  
Sometimes a bot doing cursed things.  
Sometimes a scraper that probably shouldn't exist.  
Sometimes an Android APK just to see if I can.

**I don't code to impress.**  
I code because the process itself is the reward.

<br/>

```
build → break → learn → repeat
```
